# Container Vulnerability Benchmark Suite Overview

This document provides an overview of the benchmark suites for container vulnerabilities created for different programming languages. Each suite consists of container images with known vulnerabilities, designed for evaluating the effectiveness of container vulnerability scanners.

## Benchmark Suites

*   **Python CVE Benchmark Suite:** Contains a collection of container images with various Python-related CVEs.
    *   [View Python CVE Details](Python%20Container%20Index.md)

*   **Node.js CVE Benchmark Suite:** Contains a collection of container images with various Node.js-related CVEs.
    *   [View Node.js CVE Details](Nodejs%20Container%20Index.md)

*   **Java CVE Benchmark Suite:** Contains a collection of container images with various Java-related CVEs.
    *   [View Java CVE Details](Java%20Container%20Index.md)

*   **Go CVE Benchmark Suite:** Contains a collection of container images with various Go-related CVEs.
    *   [View Go CVE Details](Go%20Container%20Index.md)
