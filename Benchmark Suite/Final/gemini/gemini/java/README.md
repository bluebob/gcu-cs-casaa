# Java Vulnerability Benchmark Set

This document details a set of 12 vulnerabilities in Java and its ecosystem, generated by Gemini for a degree research project.

## Vulnerabilities

*   **CVE-2022-22965** - "Spring4Shell"
    *   **Impact:** 9.8 (Critical) - RCE in Spring Framework data binding on JDK 9+.
*   **CVE-2026-22732** - Spring Security Header Omission
    *   **Impact:** 9.1 (Critical) - Configured HTTP security headers may not be written to the response.
*   **CVE-2025-46295** - Apache Commons Text RCE
    *   **Impact:** 9.8 (Critical) - Unsafe interpolation can lead to RCE.
*   **CVE-2025-53192** - Apache Commons OGNL RCE
    *   **Impact:** 8.8 (High) - RCE via bypassing blocklist restrictions.
*   **CVE-2024-26308** - Apache Commons Compress DoS
    *   **Impact:** 7.5 (High) - Infinite loop when reading a crafted 7Z archive.
*   **CVE-2024-47883** - OpenRefine/MIT Simile Butterfly RCE
    *   **Impact:** 9.8 (Critical) - Improper handling of `file:/` URLs can lead to RCE.
*   **CVE-2026-35229** - Oracle DB Java VM compromise
    *   **Impact:** 7.5 (High) - Unauthenticated compromise of the Java VM in Oracle DB.
*   **CVE-2026-35227** - Oracle Java SE JAXP compromise
    *   **Impact:** 7.5 (High) - Unauthenticated compromise of JAXP.
*   **CVE-2026-35228** - Oracle Java SE JSSE compromise
    *   **Impact:** 7.4 (High) - Unauthenticated compromise of JSSE.
*   **CVE-2026-21932** - Oracle Java SE AWT/JavaFX bypass
    *   **Impact:** 7.4 (High) - Security bypass in AWT/JavaFX.
*   **CVE-2025-61881** - Oracle DB Java VM compromise
    *   **Impact:** 7.7 (High) - Unauthenticated compromise of the Java VM in Oracle DB.
*   **CVE-2024-22233** - Spring Framework DoS
    *   **Impact:** 7.5 (High) - DoS via a specially crafted HTTP request.
