# C++ Vulnerability Benchmark Set

This document details a set of 12 vulnerabilities in C++ and its ecosystem, generated by Gemini for a degree research project.

## Vulnerabilities

*   **CVE-2026-34582** - Botan C++ Cryptography Library
    *   **Impact:** 9.1 (Critical) - TLS 1.3 `ApplicationData` processing issue.
*   **CVE-2026-24469** - C++ HTTP Server
    *   **Impact:** 7.5 (High) - Path Traversal.
*   **CVE-2026-34054** - vcpkg
    *   **Impact:** 7.8 (High) - Hardcoded `openssldir` path.
*   **CVE-2026-24468** - bit7z C++ Library
    *   **Impact:** 7.5 (High) - "Zip Slip" Path Traversal.
*   **CVE-2023-34399** - Boost.Serialization
    *   **Impact:** 9.8 (Critical) - Integer Overflow.
*   **CVE-2016-9840** - Boost.Beast (zlib)
    *   **Impact:** 8.8 (High) - Out-of-bounds pointer arithmetic.
*   **CVE-2018-25032** - Boost.Beast (zlib)
    *   **Impact:** 7.5 (High) - Memory corruption.
*   **CVE-2012-2677** - Boost.Pool
    *   **Impact:** 5.0 (Medium) - Integer overflow.
*   **CVE-2025-15467** - OpenSSL
    *   **Impact:** 9.8 (Critical) - Stack buffer overflow in CMS.
*   **CVE-2026-31790** - OpenSSL
    *   **Impact:** 7.5 (High) - Information disclosure in RSASVE.
*   **CVE-2026-28388** - OpenSSL
    *   **Impact:** 7.5 (High) - DoS with delta CRLs.
*   **CVE-2022-3602** - OpenSSL
    *   **Impact:** 7.5 (High) - Stack buffer overflow in X.509 verification.
