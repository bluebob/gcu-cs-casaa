# Go Vulnerability Benchmark Set

This document details a set of 12 vulnerabilities in Go and its ecosystem, generated by Gemini for a degree research project.

## Vulnerabilities

*   **CVE-2023-39325** - HTTP/2 Rapid Reset
    *   **Impact:** 7.5 (High) - A malicious HTTP/2 client can rapidly create and reset requests, causing excessive server resource consumption.
*   **CVE-2022-27664** - `golang.org/x/net/http2` DoS
    *   **Impact:** 7.5 (High) - An HTTP/2 connection can hang during closing if the shutdown process is preempted by a fatal error.
*   **CVE-2023-5528** - Kubernetes arbitrary code execution on Windows
    *   **Impact:** 8.8 (High) - A user with permissions to create Pods and PersistentVolumes on Windows nodes could escalate their privileges to admin on those nodes.
*   **CVE-2022-41720** - Path traversal on Windows (`os.DirFS`, `http.Dir`)
    *   **Impact:** 7.5 (High) - An attacker can access restricted files or Windows device files that should be outside the intended directory tree.
*   **CVE-2022-30631** - `compress/gzip` uncontrolled recursion
    *   **Impact:** 7.5 (High) - An attacker can craft a malicious GZIP archive that triggers recursive calls, leading to a panic.
*   **CVE-2022-32189** - `math/big` DoS
    *   **Impact:** 7.5 (High) - A too-short encoded message can cause a panic in `Float.GobDecode` and `Rat.GobDecode`.
*   **CVE-2023-24534** - `google.golang.org/protobuf` panic
    *   **Impact:** 7.5 (High) - Parsing a text-format message containing a potential number consisting of a minus sign followed by whitespace and no further input causes a panic.
*   **CVE-2022-23806** - `crypto/elliptic` incorrect curve check
    *   **Impact:** 9.1 (Critical) - `Curve.IsOnCurve` incorrectly returns `true` for certain invalid `big.Int` values.
*   **CVE-2024-3566** - "BatBadBut" Command Injection (Windows)
    *   **Impact:** 9.8 (Critical) - Improper escaping of command-line arguments when invoking batch files.
*   **CVE-2026-25679** - URL Parsing Validation Bypass
    *   **Impact:** 7.5 (High) - `url.Parse` insufficiently validated the host/authority component of URLs.
*   **CVE-2026-27140** - Code Smuggling & Arbitrary Code Execution
    *   **Impact:** 8.8 (High) - A vulnerability in the Go build process involving SWIG.
*   **CVE-2025-4674** - Go Command Unexpected Execution
    *   **Impact:** 7.8 (High) - The `go` command itself may execute unexpected commands when operating in untrusted VCS repositories.
