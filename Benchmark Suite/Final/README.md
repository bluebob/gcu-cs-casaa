# README

## Contents of this directory

- Benchmark Suite Spec History - history of versions of the specification used, v5.5 was the final one
- Claude - Claude generated test suite
- Claude_code_prompt_benchmark_suite.md - low level instructions generated by Claude Desktop from the Benchmark Suite Specification, subsequently fed to claude-code to generate the files
- Gemini - Gemini generated test suite
- Gemini_code_prompt_benchmark_suite.md - equivalent to the claude file but generated by Gemini Web and fed to gemini-cli



Note about the test suites: these are arranged by programming language, one directory per language. In the directories are `base-` images which are the base images used for the tests, and `CVE-` images which are the test suite items. In addition there is a [language]`-set.csv` file which is the test manifest, used to drive the Windmill (and other) automations.
